A matematikában, egészen pontosan annak az egyik ágazatában, a lineáris algebrában a ferdén szimmetrikus mátrix (úgy is ismert még, mint az aszimmetrikus vagy az antiszimmetrikus mátrix) – az egy négyzetes A mátrix, amelynek a negatív transzponáltja egyenlő magával a mátrixxal. Ez azt jelenti, hogy teljesül a következő egyenlet: A = −AT. Ha az i-ik sorban és a j-ik oszlopban az aij elem található, azaz A = (aij), akkor az aszimmetria feltétel felírható így: aij = −aji.

Önnek meg kell határoznia, hogy egy megadott négyzetes mátrix az egy ferdén szimmetrikus mátrix-e vagy sem.

Bővebben a ferdén szimmetrikus mátrixokról a Wikipédiában lehet olvasni

Bemeneti adatok: egy négyzetes mátrix egész (int) számokból álló listaként (list) megadva.

Kimeneti adatok: felelet arra a kérdésre, hogy az adott mátrix az ferdén szimmetrikus-e vagy sem logikai (bool) értékként.

Példák: 

checkio([
    [ 0,  1,  2],
    [-1,  0,  1],
    [-2, -1,  0]]) == True
checkio([
    [ 0,  1, 2],
    [-1,  1, 1],
    [-2, -1, 0]]) == False
checkio([
    [ 0,  1, 2],
    [-1,  0, 1],
    [-3, -1, 0]]) == False

Alkalmazása: A ferdén szimmetrikus mátrixok hasznosak lehetnek a vektoriális szorzatoknál, a matematikában az erőhatások irányának kiszámításakor. A mátrixok a lineáris algebra és a vektorgrafika alapját képezik.

Előfeltételek:
0 < N < 5
